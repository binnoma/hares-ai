🎯 نبذة عن المشروع

حَارِس AI هي منصة متقدمة لتحليل تطبيقات Android بشكل ساكن (Static Analysis)، مع شرح النتائج باللغة العربية باستخدام الذكاء الاصطناعي. المشروع مصمم ليكون بديلاً عربياً لأدوات مثل MobSF، مع التركيز على تجربة المستخدم العربية وتقديم توضيحات ذكية باللغة الأم.

يهدف المشروع إلى سد الفجوة في أدوات التحليل الأمني العربية، حيث تفتقر معظم الأدوات المتاحة لدعم اللغة العربية وشرح النتائج بطريقة يفهمها المطور العربي.

⚠️ تنبيه قانوني

هذه الأداة مخصصة للاستخدام التعليمي والاختبار الأمني القانوني فقط. لا يُسمح باستخدامها لتجاوز الحمايات أو أي أغراض غير قانونية. التحليل الساكن فقط - لا يتم تشغيل التطبيق.

📦 التثبيت والتشغيل

المتطلبات الأساسية

• Node.js 18+ أو Bun    • npm أو bun أو yarn

خطوات التثبيت

# استنساخ المستودع
git clone https://github.com/binnoma/hares-ai.git
cd hares-ai

# تثبيت التبعيات
bun install

# إعداد قاعدة البيانات
cp .env.example .env
bun run db:push

# تشغيل المشروع
bun run dev

💡 ملاحظة

افتح المتصفح على http://localhost:3000 بعد تشغيل المشروع. يمكنك استخدام ميزة "التحليل التجريبي" لاختبار البرنامج بدون رفع ملف APK حقيقي.

🔒 الأمان والخصوصية

حَارِس AI مصمم مع مراعاة أعلى معايير الأمان والخصوصية:

✅ تحليل ساكن فقط — لا يتم تشغيل التطبيق المحلل
✅ لا تجاوز حمايات — الأداة لا تتجاوز أي آليات حماية
✅ حذف تلقائي — الملفات المرفوعة تُحذف بعد التحليل
✅ تخزين محلي — جميع البيانات مخزنة محلياً فقط
✅ استخدام تعليمي — الأداة مخصصة للتعلم والاختبار القانوني
