# المساهمة في حَارِس AI | Contributing to Hares AI

شكراً لاهتمامك بالمساهمة في حَارِس AI! 🎉

## 🤝 كيف تساهم

### الإبلاغ عن مشاكل
1. تأكد من أن المشكلة لم يتم الإبلاغ عنها مسبقاً
2. افتح Issue جديد مع وصف واضح للمشكلة
3. أضف خطوات إعادة الإنتاج إن أمكن

### اقتراح ميزات
1. افتح Issue مع علامة `enhancement`
2. صف الميزة المقترحة وفوائدها

### إرسال تعديلات
1. Fork المستودع
2. أنشئ فرع جديد (`git checkout -b feature/amazing-feature`)
3. نفذ التعديلات
4. التزم بمعايير الكود (`bun run lint`)
5. ارسل Pull Request

## 📏 معايير الكود

- استخدام TypeScript في جميع الملفات
- اتباع نمط ESLint المعرف في المشروع
- كتابة تعليقات بالعربية للكود المعقد
- استخدام مكونات shadcn/ui الحالية

## 📜 رخصة المساهمة

بمساهمتك في هذا المشروع، فإنك توافق على أن مساهماتك ستكون تحت رخصة MIT.
